## 💡 核心心得

### 1. 关于 Prompt 技巧
> 发现给 AI 提供具体的上下文（Context）比空喊指令有效得多。

### 2. 工具对比
| 工具 | 优势 | 劣势 |
| :--- | :--- | :--- |
| Trae | 丝滑的 UI，Solo 模式很强 | 插件生态还在完善 |
| Cursor | 社区生态丰富 | 偶尔会有卡顿 |

## 🛠️ 项目实操记录
- **项目名称**：个人简历网页
- **遇到问题**：CSS 布局无法居中
- **解决办法**：发送截图给 Trae，它直接给出了 Flexbox 方案。
